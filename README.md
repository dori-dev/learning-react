# Learning React

The simple example for how to use react.

#

# How to Run Project

## Download Codes

```
git clone https://github.com/dori-dev/learning-react.git
```

```
cd learning-react
```

## Run FrontEnd

```
npm run start
```

## Run Backend

```
npm run server
```

## Open On Browser

Index: [localhost:3000](http://localhost:3000/)<br>
